# AI Sentiment Analysis Web Application using IBM Watson NLP and Flask

## Overview

This project was developed as part of the **IBM Generative AI Engineering** course and demonstrates the complete development lifecycle of an AI-powered sentiment analysis web application.

The application allows users to submit textual input through a web interface and receive sentiment analysis results generated by a pre-trained IBM Watson NLP model. The system integrates Natural Language Processing (NLP), web development, software testing, and code quality assessment into a single end-to-end project.

---

## Features

* AI-powered sentiment analysis using IBM Watson NLP
* Flask-based web application
* REST API integration
* Modular Python package structure
* Unit testing with Python's `unittest`
* Static code analysis using `Pylint`
* Client-server communication using JavaScript and Flask
* Web-based user interface for sentiment classification

## Functionality

The application follows the workflow below:

1. The user enters text through the web interface.
2. JavaScript sends the input to the Flask server.
3. Flask forwards the text to the sentiment analysis module.
4. The module communicates with the IBM Watson NLP service.
5. The sentiment label and confidence score are returned.
6. The result is displayed dynamically in the browser.

This project was originally developed and executed within the **IBM Skills Network Lab** environment.
